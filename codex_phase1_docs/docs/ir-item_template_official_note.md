# ir-item_template_official_note.md

## このファイルの目的
Codex が「最新公式テンプレート」と「手元の参考サンプル」を混同しないように整理する。

## 正本（source of truth）
`ir-item.csv` の正本ヘッダは、**item Robot アカウントからダウンロードした最新の公式テンプレート**とする。

公開マニュアル上でも、テンプレートは item Robot の画面からダウンロードする前提で案内されている。
また、所持店舗によって出力される項目と出力されない項目があるため、環境依存差分がありうる。

## 参考として同梱したファイル
### 1. `samples/ir-item_reference_header_from_sample.csv`
- 手元の現行実データサンプルからヘッダだけを抜き出したもの
- 参考用
- 正本ではない

### 2. `samples/ir-item_current_sample.csv`
- 現行運用サンプル
- 列の使われ方や実データ傾向の確認用

## Codexへの指示
1. まず `samples/ir-item_reference_header_from_sample.csv` を使って実装方針を作る
2. もし実際の最新公式テンプレートが別途提供されたら、そちらを優先してヘッダ順・列名を合わせる
3. 手元サンプルと公式テンプレートに差分がある場合、公式テンプレートを優先する
4. どのヘッダを正本として採用したか README に明記する

## 補足
公開マニュアルから分かること:
- テンプレートは「商品管理＞CSVダウンロード＞テンプレートダウンロード」から取得する
- `ir-item.csv` はテンプレートダウンロード対象の一つ
- 出力列は環境依存差分がありうる
- 新規商品は `ir-item.csv` からインポートするのが基本
